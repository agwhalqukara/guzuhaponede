The Graph (GRT) is a token that powers The Graph, a decentralized indexing protocol used by blockchains such as Ethereum.  Similar to how search engines such as Google index the Internet, Graph protocol indexes blockchain data, grouping it into subgraphs on open application programming interfaces (API) that anyone can query via GraphQL, Graph’s programming language.  Subgraphs can then be formed into a global graph of all the public information.
